Module axi_iw_downsizer_intf

Parameters

NUM_IDS: int unsigned

Number of IDs that should be active on the master port side. E.G.: If chosen as 32'd3 ther will be the IDs: 0, 1, 2. This has to satisfy: $clog2(NumIds) <= AxiIdWidthMst

MAX_TRANS: int unsigned

Maximum number of transactions in flight

AXI_ID_WIDTH_SLV: int unsigned

AXI4+ATOP ID width of the slave port

AXI_ID_WIDTH_MST: int unsigned

AXI4+ATOP ID width of the master port

AXI_ADDR_WIDTH: int unsigned

AXI4+ATOP address width of both ports

AXI_DATA_WIDTH: int unsigned

AXI4+ATOP data width of both ports

AXI_USER_WIDTH: int unsigned

AXI4+ATOP user width of both ports
