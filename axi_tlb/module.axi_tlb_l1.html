Module axi_tlb_l1

Internal module of axi_tlb: L1 translation table.

Parameters

InpAddrWidth: int unsigned

Width of addresses in input address space

OupAddrWidth: int unsigned

Width of addresses in output address space

NumEntries: int unsigned

Number of entries in translation table

CfgAxiAddrWidth: int unsigned

Address width of configuration AXI4-Lite port

CfgAxiDataWidth: int unsigned

Data width of configuration AXI4-Lite port

lite_req_t: type

Request type of configuration AXI4-Lite slave port

lite_resp_t: type

Response type of configuration AXI4-Lite slave port

res_t: type

Type of translation result. Must have a single-bit field hit and an addr field as wide as the output address.

inp_addr_t: type

Derived (=do not override) type of input addresses

oup_addr_t: type

Derived (=do not override) type of output addresses

InpPageNumWidth: int unsigned

OupPageNumWidth: int unsigned

InpPageNumBytes: int unsigned

InpPageNumBytesAligned: int unsigned

OupPageNumBytes: int unsigned

OupPageNumBytesAligned: int unsigned

FlagBytes: int unsigned

FlagBytesAligned: int unsigned

EntryBytesAligned: int unsigned

RegNumBytes: int unsigned

AxiReadOnly: entry_bits_t [NumEntries-1:0]

Ports

clk_i: input logic

Rising-edge clock of all ports

rst_ni: input logic

Asynchronous reset, active low

test_en_i: input logic

Test mode enable

wr_req_addr_i: input inp_addr_t

Write request input address

wr_req_valid_i: input logic

Write request valid

wr_req_ready_o: output logic

Write request ready

wr_res_o: output res_t

Write translation result

wr_res_valid_o: output logic

Write translation result valid

wr_res_ready_i: input logic

Write translation result ready

rd_req_addr_i: input inp_addr_t

Read request input address

rd_req_valid_i: input logic

Read request valid

rd_req_ready_o: output logic

Read request ready

rd_res_o: output res_t

Read translation result

rd_res_valid_o: output logic

Read translation result valid

rd_res_ready_i: input logic

Read translation result ready

cfg_req_i: input lite_req_t

Configuration port request

cfg_resp_o: output lite_resp_t

Configuration port response

Types

inp_page_t

Page number in input address space

oup_page_t

Page number in output address space

entry_t

Translation table entry with 4 KiB page granularity
